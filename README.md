
# test01 lib sagui project

You must installed in linux ubuntu distro and derivated

- build-essentials
- cmake
- git

Commands for linux

```bash
sudo apt-get install build-essentials cmake git
git clone https://github.com/sirlordt/test01_libsagui.git
cd test01_libsagui
mkdir build
cd build
cmake ..
cmake --build .
```
